# FinSim - Financial Simulation Platform

A monorepo containing the backend API and frontend application for financial simulation and backtesting.

## Project Structure

```
├── backend/      # Python Flask API for backtesting and simulation
├── frontend/     # Frontend application (Coming soon)
└── .gitignore    # Monorepo gitignore
```

## Backend

The backend provides APIs for:

-   Stock data fetching
-   Backtesting trading strategies
-   Portfolio optimization
-   Trading simulation

See [backend/README.md](backend/README.md) for more details.

## Getting Started

### Backend Setup

```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
python run.py
```
